# Revista Aqua

Projeto desenvolvido como teste para vaga de Front-End.

## Tecnologias
- Next.js
- TypeScript
- SCSS Modules
- Lucide Icons

## Funcionalidades
- Listagem de revistas
- Destaque da edição principal
- Visualização de PDF em modal (experiência tipo livro)
- Download de revista
- Compartilhamento
- Responsivo

## Como rodar

- npm install
- npm install sass lucide-react
- npm run dev
